This Python project scapes The [What's On Glasgow page] (https://www.whatsonglasgow.co.uk/events/) for events and saves the event title, date, location, description, and a link to the full event page to a CSV file.

It uses 'requests' and 'beautiful soup' for scraping.

## This project can be used for:
- Finding fun things to do in Glasgow!


## To run this program:
Make sure you have Python 3 installed on your machine and download the requests and beautiful soup libraries by entering *pip install requests beautifulsoup4* in your terminal.
